## Node.js 与 Jupyter 环境

Node.js 与 NPM 是前后端与许多工具链的基础。Node.js 与 NPM 的安装与使用指南可参考示例教程（[https://blog.csdn.net/weixin\_62818371/article/details/146175470），推荐使用](https://blog.csdn.net/weixin_62818371/article/details/146175470），推荐使用) Node.js v22.19.0 LTS。Jupyter 环境中，内核管理常用命令如下：

```bash
# 安装 ipykernel
pip install ipykernel

# 注册自定义内核
python -m ipykernel install --user --name myenv

# 注册并自定义显示名称
python -m ipykernel install --user --name yourname --display-name yourname
python -m ipykernel install --user --name AI --display-name AI
```
